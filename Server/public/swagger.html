📜 Documentation de l'API

Utilisateurs

🔹 Récupérer tous les utilisateurs

GET /users

🔹 Créer un nouvel utilisateur

POST /users

🔹 Récupérer un utilisateur par ID

GET /users/{id}

🔹 Mettre à jour un utilisateur

PUT /users/{id}

Quiz Questions

🔹 Récupérer toutes les questions

GET /quiz-questions

🔹 Ajouter une question

POST /quiz-questions

🔹 Récupérer 10 (autant que voulue) questions aléatoires

GET /quiz-questions/random?limit=10 (ajustable)

🔹 Récupérer une question par ID

GET /quiz-questions/{id}

Scores

🔹 Récupérer tous les scores

GET /scores

🔹 Récupérer les scores d'un user pr son ID

GET /scores/user/{id}

🔹 Ajouter un score

POST /scores

🔹 Récupérer un enregistrement de score par ID

GET /scores/{id}

Leaderboard

🔹 Récupérer le classement International

GET /leaderboard

🔹 Récupérer le classement d'un pays

GET /leaderboard/country/{country}
