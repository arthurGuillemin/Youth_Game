📜 Documentation de l'API

👤 Utilisateurs

🔹 Récupérer tous les utilisateurs

GET /users

🔹 Récupérer un utilisateurs avec son id

GET /users/{id}

🔹 Créer un utilisateur

POST /users

🔹 Mettre à jour un utilisateur

PUT /users/{id}

❓ Quiz Questions

🔹 Récupérer toutes les questions

GET /quiz-questions

🔹 Ajouter une question

POST /quiz-questions

🔹 Récupérer un nombre de questions

GET /quiz-questions/random?limit=10 (par exemple)

🔹 Récupérer une question avec son id

GET /quiz-questions/{id}

🔹 Update une question avec son id

PUT /quiz-questions/{id}

🔹 Delete une question avec son id

DELETE /quiz-questions/{id}

⭐ Scores

🔹 Récupérer tout les scores

GET /scores

🔹 Récupérer un score avec son id

GET /scores/{id}

🔹 Récupérer tout les scores d'un user avec son id

GET /scores/{id}

🔹 Ajouter un score

POST /scores

🔹 Update un score avec son id

PUT /scores/{id}

🔹 Delete un score avec son id

Delete /scores/{id}

🏆 Classement

🔹 Classement général

GET /leaderboard

🔹 Classement national

GET /leaderboard/country/{country}

🔹 Classement international

GET /leaderboard/countries

🎮 Parties

🔹 Récupérer toutes les parties

GET /parties

🔹 Récupérer une partie par son id

GET /parties

🔹 Récupérer les usernames des participants avec l'id de la partie

GET /parties/{id}/users

🔹 Créer une partie

POST /parties

🔹 Ajouter un joueur a une partie

POST /parties/{id}/add

body exemple : { "user_id": "ddfefa2d-bc06-4fe8-a681-3bc6361d9ba9" }

🔹 Suprr un joueur d'une partie

POST /parties/{id}/remove

body exemple : { "user_id": "ddfefa2d-bc06-4fe8-a681-3bc6361d9ba9" }

🔹 Update une partie

PUT /parties

🔹 Supprimer une partie

DELETE /parties/{id}
